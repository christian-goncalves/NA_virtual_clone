```markdown
# TAREFA DE DESENVOLVIMENTO — ETAPA 1
## Projeto: NA Virtual Clone

Utilize como **fonte de verdade obrigatória** os arquivos do projeto:

---

# CONTEXTO DO PROJETO

O sistema irá reconstruir uma página pública que exibe **reuniões virtuais de Narcóticos Anônimos**.

A aplicação será construída com:

- Laravel
- Blade
- Tailwind
- Alpine.js
- Redis
- Queue
- Scheduler
- DomCrawler

A fonte oficial dos dados será:

https://www.na.org.br/virtual/

As reuniões serão **sincronizadas e armazenadas localmente no banco de dados** antes de serem exibidas na página.

---

# ETAPA ATUAL DO DESENVOLVIMENTO

Estamos executando a **ETAPA 1 — Estrutura de dados**.

Objetivo:

Criar a base onde as reuniões serão armazenadas.

Conforme definido na arquitetura do projeto, a tabela principal será:

```

virtual_meetings

```

Esta tabela armazenará as reuniões coletadas do site oficial.

---

# TAREFAS A EXECUTAR

Criar a migration:

```

create_virtual_meetings_table


virtual_meetings


## 2️⃣ Criar Model

Criar o model:

```
VirtualMeeting
```

Local:

```

```

O model deve incluir:

* `$fillable`
* `$casts` para campos JSON
* `$casts` para campos booleanos
* `$casts` para datas
